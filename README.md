# user-account-auth-system
User Account CRUD with a session-based auth system

![logs](https://github.com/osag1e/user-account-auth-system/blob/main/logs.png)


## Makefile commands are:
```
make build-api
make run
make lint
make test
make cyclomatic
make clean
```


## Project environment variables
```
HTTP_LISTEN_ADDRESS=
DB_HOST=
DB_PORT=
DB_USER=
DB_PASSWORD=
DB_NAME=
DB_SSLMODE=
```
